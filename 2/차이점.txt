텍스트 파일은 문자 데이터를 저장하며, 사람이 읽을 수 있는 형태이며, 이진파일은 바이트 단위로 저장되며, 정수, 실수, 문자열 등 원시 형태로 저장함
이진 파일: 데이터를 있는 그대로 읽고 씀 (숫자 123 저장 -> 가공 X 숫자 123이 그대로 저장)

예시: 숫자 123을 이진 파일에 저장하면 컴퓨터는 그대로 123이라는 값을 메모리에서 디스크로 복사함. 정수 123은 4바이트를 차지하고 01111011처럼 바이트로 저장됨
다시 읽을 때도 바이트 그대로 읽어서 123으로 변환함. 그래서 정확히 123이라는 값을 읽고 쓸 수 있음
크기가 작고 빠르지만 사람이 파일 내용을 직접 보면 이상한 문자나 기호로 보임

텍스트 파일: 숫자를 문자로 변환 후 사용 (숫자 123 -> 문자 '1', '2', '3'으로 저장)

예시: 숫자 123을 텍스트 파일에 저장하면 '1', '2', '3' 이렇게 세 개의 문자로 변환되어 저장됨. 각 문자는 ASCII나 UTF-8 같은 인코딩으로 저장됨
메모장으로 열어보면 "123"이라고 보이지만 실제로는 문자 '1', '2', '3'이 각각 저장된 것임
사람이 직접 읽을 수 있지만 파일 크기가 커지고 숫자를 다시 계산에 사용하려면 문자열을 숫자로 변환하는 과정이 필요함
